# Ai Translation Helper 601

A CLI tool that translates text between languages using a neural machine translation model, with support for multiple languages and customizable translation settings.

## Features

- Support for over 100 languages
- Customizable translation settings
- Batch processing of text files
- Interactive mode for real-time translation
- Model versioning and update support

## Installation

```bash
# Clone the repository
git clone https://github.com/{username}/ai-translation-helper-601.git
cd ai-translation-helper-601

# Install dependencies
pip install -r requirements.txt
```
